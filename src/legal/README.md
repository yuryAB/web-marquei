# Documentos legais

Este diretório reúne os arquivos JSON usados como fonte de verdade para as páginas públicas de termos de uso e política de privacidade deste projeto.

## Arquivos

- `privacy-policy.json`: conteúdo da política de privacidade
- `terms-of-use.json`: conteúdo dos termos de uso

## Como funciona

As páginas públicas em `legal/privacy-policy/` e `legal/terms-of-use/` carregam estes arquivos e renderizam o conteúdo no site.

## Atualização

Para atualizar os documentos:

1. edite o JSON correspondente;
2. revise o texto e a data de atualização;
3. publique novamente a landing.
